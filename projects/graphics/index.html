Modern Graphics Programming

This workshop is about working through and completing the book Learning Modern 3D Graphics Programming by Jason L. McKesson (2012). The book is available for free online. All programmers and technical minded people are welcome to join every week. There will be weekly sessions, every Friday. Bring a computer. If you have a mac with something newer than macos 10.13, you will need to use a virtual machine with linux to follow this course.
