Modern Graphics Programming

This workshop is about working through and completing the book Learning Modern 3D Graphics Programming by Jason L. McKesson (2012). The book is available for free online. All programmers and technically-minded people are welcome to join every week.

There will be weekly sessions, every Friday. Bring a computer. If you have a mac with an operating system newer than macos 10.13, you will need to use a virtual machine with linux to follow this course.

This course might cover some of the contents of 23DPRO101 3D-Programming, a course which is covered in the third year, spring semester, for games technology students.

The book consists of six parts and 17 chapters. One chapter is completed each week. The course starts 12. January 2024 and ends the 3. May 2024.
